How it works

1Press the hotkey

Trigger the capture overlay anytime: Ctrl + Alt + H.

2Drag to select the text

Draw a rectangle around the words you want. Tight selections improve accuracy.

3OCR reads what’s on your screen

ClipOCR preprocesses the image and runs Tesseract OCR automatically.

4Text is copied to clipboard

Paste into Excel, Notion, Google Docs, chat apps—anywhere.

For best results: zoom PDFs to 150–250% before capture.
